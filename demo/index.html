iostap

iostap is a plugin designed to mimic the behaviour of tapping a button in the iOS ecosystem with a great degree of precision, affording developers who build hybrid web apps a greater experience for their users. iostap is a super-lightweight library, weighing in at about 1kb, and doesn't add unecessary overhead client-side.

Download the source files from the build directory, or use Bower.

$ bower install iostap

Active state

Button

The active state should be removed once the user moves their pointer a significant distance from the button. This cancels the"iostap"event. Bringing the finger back to the button will re-instate the active class.

.button {...}

// Active styles
.button.__active {...}
